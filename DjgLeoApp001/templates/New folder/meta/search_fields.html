search_fields

Columns from demo: blog__name (not shown, but search "first" or "second" to find entries being filtered by this setting)

Additional database fields to query for global searches can be specified with this setting.

Normally you would simply back each of your columns with the correct list of database fields, but if valid searchable fields don't actually appear in the columns, you can have these extra fields appended to the list of searchable items without lying about the fields in your column specification.
